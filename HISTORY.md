## v0.1

Initial release.

**Please use with caution - testing suite is currently extremely limited**

### Problem Types Supported

- simple p-median (including setting of maximum cost cutoff)
- standard p-median (including setting of maximum cost cutoff)
- p-center
- maximal covering location problem

### Solvers Supported

- brute force
- greedy
- GRASP (greedy randomised adaptive search procedure)
